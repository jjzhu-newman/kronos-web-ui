## 设置方法

1. 访问仓库页面：https://github.com/jjzhu-newman/kronos-web-ui
2. 点击 Settings（设置）
3. 找到 "Topics" 部分
4. 点击 "Add topics"
5. 搜索并添加上述标签

推荐添加的核心标签：
```
machine-learning, time-series, forecasting, finance, pytorch, flask, web-ui, data-visualization, stock-market, cuda
```
